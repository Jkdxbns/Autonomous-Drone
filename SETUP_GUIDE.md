## Troubleshooting Common Issues

### "Package not found" after copying files
```bash
flutter clean
flutter pub get
```

### Android build fails
- Ensure you copied `android/app/build.gradle.kts`
- Ensure you copied `android/app/src/main/AndroidManifest.xml`
- Run `flutter clean` and rebuild

### Permissions not working
- Check `AndroidManifest.xml` was copied correctly
- Reinstall the app (uninstall first)
- Manually grant permissions in Android Settings
